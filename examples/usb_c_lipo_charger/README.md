# USB-C LiPo Charger

A standalone USB-C LiPo battery charger with power path management.

## Specifications

- **Charger:** MCP73831 (500mA)
- **Protection:** TP4056 with DW01 protection IC
- **Connector:** USB-C (USB 2.0 mode)
- **Dimensions:** 25×15mm, 2-layer
- **Features:** Charge status LED, battery connector (JST-PH), enable jumper
