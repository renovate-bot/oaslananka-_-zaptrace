# STM32 RS-485 Node

Industrial STM32-based RS-485 communication node with isolated power.

## Specifications

- **MCU:** STM32G070 (TSSOP-20)
- **Transceiver:** MAX3485 (3.3V RS-485)
- **Isolation:** ISO7741 digital isolator + isolated DC-DC
- **Power:** 12-24V DC input → LM2596 buck converter
- **Dimensions:** 50×40mm, 2-layer
- **Features:** Termination jumper, TVS protection, LED indicators
