# nRF52840 BLE Sensor

Ultra-low-power Bluetooth sensor node with nRF52840.

## Specifications

- **MCU:** nRF52840 (QFN-48)
- **Sensors:** LSM6DSO (IMU), SHT40 (temp/humidity)
- **Battery:** 100mAh LiPo, charging via BQ25120
- **Dimensions:** 35×25mm, 4-layer
- **Features:** USB-C, RGB LED, SWD programming, coin cell backup
